Canvas

Canvas

  • Ein Tag, mit dem Sie auf einem starken und präzisen sowie auch niedrigem Niveau zeichnen können.
  • Grafikfunktionen werden automatisch verwendet
  • Rendering ist optimiert
  • Canvas besteht aus Context
  • Der Context ermöglicht ein Zeichnen
  • Zeichenobjekte:
    • Rechtecke
    • Linien
    • Kurven / Kreise
    • Bezierkurven
    • Bilder
    • Text

Verwenden des Canvas-Elements

  • <canvas id="game"> </canvas>
  • var canvas = document.getElementById(“game”);
  • var c = canvas.getContext(‘2d’);

Zeichne ein Rechteck

  • c.fillRect(x, y, Breite, Höhe)
  • c.rect(x, y, Breite, Höhe)
  • c.fillStyle = 'rgba(0, 255, 125, 0.3)' - definiert den zu verwendenden Stil.

* c.fillRect

* c.rect

Linien

  • Wir beginnen die Zeichnung mit: – c.beginPath()
  • Die Zeichnung beenden wir mit: – c.stroke() / c.fill()
  • c.moveTo(x,y) – Bewegt "Hand" zur Position
  • c.linieTo(x,y) – Verbindet den aktuellen Punkt mit Punkt x, y mithilfe einer Linie
  • c.strokeStyle – definiert den Strichzeichnungsstil
c.beginPath();
c.moveTo(200, 100);
c.lineTo(450, 150);
c.lineTo(450,50);
c.strokeStyle = “red
c.stroke()

Gradient

  • var gradient = c.createLinearGradient(x0, y0, x1, x1);
  • x0,y0 – Anfangskoordinaten des Gradienten
  • x1,y1 – endgültige Gradientenkoordinaten
    • gradient.addColorStop("0", "magenta");
    • gradient.addColorStop("0.5", "blue");
    • gradient.addColorStop("1.0", "red");
  • Kann zu fillStyle oder StrokeStyle Attribute hinzugefügt werden.
    • c.fillStyle = gradient

Kurve

  • c.arc(x, y, Radius, Anfangswinkel, endWinkel)
  • (Das optionale 6. Argument, wenn es true ist, bedeutet Richtung gegen den Uhrzeigersinn)

Text

  • c.font = "30px Arial";
  • c.fillText("Hello World", 10, 50);
  • oder
  • c.font = "30px Arial";
  • c.strokeText("Hello World", 10, 50);

clearRect

  • Löscht den Inhalt innerhalb des Rechtecks
  • c.clearRect(0, 0, Breite, Höhe)

requestAnimationFrame

  • Aktiviert Animation innerhalb des Canvas-Elements
  • Wird aufgerufen, wenn der Code erneut ausgeführt werden muss Plotten
  • Wird bei Bedarf vom Browser aufgerufen
  • Die Idee ist, etwa 60 Anrufe pro Sekunde zu haben
  • function animate() {
            requestAnimationFrame(animate)
         }
